## 核心思路
完型近半空格靠**逻辑关系**判断：
- 因果：so / therefore / as a result / because / since
- 转折：but / however / yet / nevertheless / although
- 递进：moreover / furthermore / besides / what's more
- 对比：while / whereas / in contrast

## 解题流程
1. 跳读上下句，找逻辑关系词
2. 同向选近义词，反向选反义词
3. 代入复读确认通顺

## 易错点
- 把 however 误读成 moreover
- 单看一句，忽略段落逻辑
